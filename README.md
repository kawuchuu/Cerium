# ProJshMod

A Discord bot that's in very early stages. Expect bugs and crashes.

# Programs and Packages Required
- Dependencies
    - `discord.js`
    - `opusscript` (you should use `node-opus` if possible)
    - `ytdl-core`
- Programs
    - Node.JS
    - A terminal client
    - FFmpeg (required for `!play` command)
